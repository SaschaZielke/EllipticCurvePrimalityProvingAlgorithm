Zur Benutzung des Programmes:
- Es wird empfohlen das mitgelieferte Zip-Verzeichnis als Projekt in der 
  Eclipse IDE zu importieren. In diesem Fall sollte der Code direkt ausführbar
  sein. 
- Bevorzugt man es, das Zip-Verzeichnis zu entpacken und die Quellcode-Dateien
  manuell in seine Entwicklungsumgebung zu laden, sollte man darauf achten,
  dass alle Klassendateien im selben Ordner (im gleichen Package liegen). 
  Außerdem sollte man die 8 weiteren Bibliotheken als externe .jar-Files
  einbinden, die im Ordner lib hinterlegt sind. Dies sind die nötigen Files,
  die zur Verwendung der beiden Bibliotheken JAS und big-math benötigt werden.
  (zur näheren Information wird auf die Dokumentation und Einrichtungshinweise
   der jeweiligen Bibliothek verwiesen. Die entsprechenden Links finden sich
   in Kapitel 4 der Masterarbeit.)
- Das Programm kann man durch Ausführen des main-Programmes in der Klasse
  EllipticCurvePrimalityProving starten. Dort kann man entsprechend anpassen
  für welche Zahl man gerne überprüfen möchte, ob sie prim ist oder nicht.
- Die Dokumentation des Programmes befindet sich mittels JAVA-Docs im Quellcode.
- Das Programm zur Ermittelung der mittleren Laufzeit von Kapitel 4 der Arbeit
  befindet sich ebenfalls (auskommentiert) im Hauptprogramm in der Klasse
  EllipticCurvePrimalityProving.
- Sollten weitere Fragen zur Benutzungauftreten, können Sie sich gerne an mich
  wenden.
